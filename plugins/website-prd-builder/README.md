# website-prd-builder

워크숍용 웹사이트/웹서비스 PRD 빌더 플러그인입니다. `/website-prd` 슬래시 커맨드 하나로 PRD 프레임 작성부터 팀별 폴더 구조 생성, GitHub 푸시까지 자동화합니다.

## 사용법

Claude Desktop 앱의 **Code 탭**에서 아래 커맨드를 입력하세요.

```
/website-prd
```

커맨드를 실행하면 몇 가지 질문에 답한 뒤 PRD 문서와 팀별 작업 폴더가 자동으로 생성됩니다.

## 생성되는 폴더 구조

커맨드를 실행하면 다음 5개의 팀 폴더가 자동으로 생성됩니다.

```
.
├── 01-기획/
├── 02-디자인/
├── 03-CX/
├── 04-마케팅/
└── 05-전략/
```

## 5개 팀 구조

| 폴더 | 팀 | 역할 |
|---|---|---|
| `01-기획` | 기획팀 | 서비스 컨셉, 기능 정의, PRD 본문 작성 |
| `02-디자인` | 디자인팀 | 와이어프레임, UI/UX, 브랜드 비주얼 |
| `03-CX` | CX팀 | 고객 여정, 온보딩, 페인포인트 설계 |
| `04-마케팅` | 마케팅팀 | 타겟 설정, 채널 전략, 런칭 플랜 |
| `05-전략` | 전략팀 | BM, 경쟁 분석, 성장 지표 정의 |

각 폴더에는 해당 팀이 작업해야 할 기본 문서 템플릿이 함께 생성됩니다.
